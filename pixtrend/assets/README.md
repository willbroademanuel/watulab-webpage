# PixTrend Assets

This folder is reserved for PixTrend product images and media.

## Add Your Images Here

Drop your images into this folder and reference them in `index.html` as:

```html
<img src="assets/your-image-name.jpg" alt="Description of the image">
```

## Naming Conventions

- Use **lowercase letters only**
- Use **hyphens** instead of spaces (e.g., `ai-demo-before.jpg`)
- No special characters: only letters, numbers, and hyphens
- Keep file sizes under 500KB for best performance (compress at https://squoosh.app)

## Recommended Image Sizes

| Usage | Recommended size |
|---|---|
| Hero / before-after demo | 1260×750px |
| Feature card mini-slider | 600×400px |
| Portrait / product shot | 600×800px |
